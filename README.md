# AI-Powered UGC Ad Content Pack

## Project Overview
This project was created as part of the Prompt Engineering Internship by Future Interns.

The goal of this project is to use AI tools like ChatGPT to create high-converting UGC-style ad content for a local salon business.

## Business Chosen
GlowUp Beauty Salon

## Features
- UGC-style ad hooks
- Instagram Reel scripts
- CTA-focused content
- AI-generated captions
- Reusable prompts

## Tools Used
- ChatGPT
- VS Code
- GitHub

## Project Structure
- prompts/
- outputs/
- assets/

## Outcome
This project demonstrates how AI can help businesses create engaging social media marketing content quickly and efficiently.
